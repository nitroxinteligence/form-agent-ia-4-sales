# ETAPA INICIAL (etapa obrigatória)

1. Logotipo (IA FOUR SALES) no topo
2. Card com dois campos e com placeholder nos campos de preenchimento:
- Digite seu nome completo
- Me fala teu email?
- Escolha a função principal do seu Agente
(Botão select com as opções: Agente SDR, Agente Vendas, Agente Customer Success, Agente Suporte, Outro)
- Se o usuário selecionar "Outro" abre um campo para ele digitar

Botão: "Continuar" 
Obs: o botão apenas libera para clicar quando o usuário preenche tudo.

# SEGUNDA ETAPA (etapa obrigatória)

- Título: "Perguntas específicas para o treinamento do Agente"

1. Nome do Agente (Qual o nome o usuário quer dar para o Agente, é com este nome que o Agente irá se apresentar, se comportar, etc)
2. Descrição breve do Agente (Ex: Agente SDR responsável por interagir com os clientes potencialmente interessados em adquirir o produto.)
3. Quais regras invioláveis que o Agente deve seguir? (Ex: não solicitar documentos, não fornecer informações confidenciais, etc)
4. Qual tom de voz do Agente? (Ex: informal, formal, etc)
5. Quais comportamentos o Agente deve adotar e/ou evitar? (Ex: ser amigável, respeitoso, etc. Jamais destratar o lead, etc.)
6. Há algum foco principal específico para o Agente? (Ex: 100% do foco em qualificar e quebrar objeções do lead.)
7. Qual o principal objetivo que o Agente deve alcançar?
8. Como o Agente deve lidar com perguntas que não sabe responder? (Ex: transferir para atendente humano, etc.)

# SEGUNDA ETAPA (etapa obrigatória)

- Título: "Agora vamos tratar sobre o seu produto/serviço"

1. Nome do produto/serviço (Ex: produto XYZ, serviço ABC)
2. Descrição detalhada
3. Principais características e benefícios
4. Diferenciais competitivos
5. Público-alvo

# TERCEIRA ETAPA (etapa opcional)

- Título: "Processo de vendas: qual a sua estratégia?"
- Texto: "Observação: responda apenas se você acredita que o seu Agente deve VENDER"

1. Processo de vendas atual (Funil de vendas)
2. Principais objeções enfrentadas nas vendas (Ex: preço, qualidade, etc)
3. Como o Agente deve lidar com essas objeções? (Ex: explicar a situação, oferecer uma solução alternativa, etc)
4. Quais são as metas de vendas para o Agente? (Ex: vender 10 leads no mês, etc)
5. Como o Agente deve se comunicar com os leads? (Ex: telefone, e-mail, chat, etc)

# QUARTA ETAPA (etapa opcional)

- Título: "Agora forneça detalhes que possam ajudar a personalizar seu Agente"

1. Você tem casos de sucesso relevantes? (Ex: um cliente que comprou o produto/serviço, um lead que foi convertido, etc) Se sim, por favor, descreva-os detalhadamente.
2. Script de vendas, atendimento, etc... (Ex: um script de vendas bem-sucedido, um script de atendimento ao cliente, etc)
3. Acesso à base de conhecimentos do produto/serviço (Ex: Tudo sobre seu produto/serviço, FAQ, etc)
- Texto de apoio: "Aqui você pode inserir arquivos para inserir na base de conhecimento do seu Agente"

# QUINTA ETAPA (etapa opcional)

- Título: "Integração com Sistemas Externos"
- Texto: "Observação: responda apenas se você acredita que o seu Agente deve se comunicar com sistemas externos, caso você tenha contratado nosso CRM, preencha estas informações da mesma forma."

1. Sistemas externos que o Agente deve integrar (Ex: CRM, HubSpot, Salesforce, etc)
2. Quais são as funcionalidades principais que o Agente deve ter acesso? (Ex: criar leads, atualizar informações do cliente, etc)

# SEXTA ETAPA (etapa opcional)

- Título "Integração com sistema de agendamento"
- Texto: "Aqui você pode integrar o seu Agente com um sistema de agendamento, como o Calendly, Google Calendar e Outlook Calendar, para que ele possa agendar reuniões com os leads. Lembrando: caso tenha contratado o nosso CRM, você terá o Calendário dentro do CRM também."

1. Ferramenta de agendamento utilizada (Ex: Calendly, Google Calendar, Outlook Calendar, etc)
2. Horários disponíveis para agendamento (Ex: 9h às 18h, segunda à sexta-feira, etc)
3. Informações necessárias para agendamento (Ex: o Agente deve solicitar nome e e-mail para poder agendar)

# SÉTIMA ETAPA (etapa obrigatória)

- Título "Comunicação do Agente"
- Texto "Configure as regras de comunicação e operação do seu Agente."

1. Como o agente deve operar em cada etapa?
2. Você deseja que o Agente realize follow-ups? Se sim... Quais regras o agente deve seguir para fazer follow-up?
3. Como o agente deve enviar lembretes? (Ex: para reuniões agendadas o Agente deve enviar um lembrete 15 minutos antes da reunião.)
4. Qual é o tempo de resposta esperado do Agente?
